WP_FAIL2BAN_PLUGIN_LOG_AUTH
---------------------------

.. versionadded:: 4.2.0

To enable logging plugin "Auth" class messages, add the following to ``wp-config.php``:

.. code-block:: php

	define('WP_FAIL2BAN_PLUGIN_LOG_AUTH', true);
